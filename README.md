CALCULATOR CLI - A Command-line calculator built with Python that performs both basic and complex mathematical operations.

FEATURES:

BASIC OPERATIONS:

1) Addition
2) Subtraction
3) Multiplication
4) Division
5) Exponentiation
6) Remainder (Modulo)
7) Percentage Calculation

ADVANCE OPERATIONS:

1) Square Root Calculator
2) Factorial
3) Average of Multiple Numbers
4) Maximum from a List of Numbers
5) Minimum from a List of Numbers
6) Quadratic Equation Calculator

INPUT VALIDATION:

The calculator includes error handling for:
1) Invalid Numeric Inputs
2) Division by Zero
3) Negative Values where Mathematically Invalid

TECHNOLOGIES USED:
1) Python 3
2) Built-in math module

FUTURE IMPROVEMENTS:
1) History of Calculations
2) Scientific Calculator (Trigonometric, Logarithmic)
3) Unit Conversion
